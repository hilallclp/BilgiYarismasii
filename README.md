# Bilgi Yarışması Projesi

Bu proje, Java kullanılarak geliştirilmiş bir **bilgi yarışması uygulamasıdır**.  
Projede katılımcılar soruları cevaplayarak yarışır ve sunucu/istemci mimarisi ile çalışır.

## Proje Özeti

- **Server-Client yapısı**: Sorular sunucudan istemcilere gönderilir.
- **GUI (Grafiksel Arayüz)**: Kullanıcıların cevaplarını kolayca girmesi sağlanır.
- **Joker ve Oyun Mekanikleri**: Oyun esnasında joker kullanımı ve puanlama.

## Kullanılan Teknolojiler

- Java 11+
- Swing GUI
- Eclipse IDE (proje ayarları mevcut)


## Kurulum ve Çalıştırma

1. `src/` klasörünü Eclipse veya herhangi bir Java IDE ile açın.  
2. `resimler/` klasörünü proje ile aynı dizinde tutun.  
3. `ProgramServer` ve `YarismaciClientGUI` sınıflarını çalıştırın.  
4. Oyuna başlamak için önce server başlatılmalı, ardından istemciler bağlanmalı.
